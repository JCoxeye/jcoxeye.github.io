JCoxeye's Leitmotif Index:
'Quiet'

Summary


('Quiet', as heard in Quiet Water)

A soft, rising arpeggio can be used to instill a quiet or tranquil feeling into a song.
This page, rather than cataloging a particular leitmotif per se, instead aims to catalogue the usage of this particular musical convention throughout the Undertale and Deltarune soundtracks (and more!).

Appearances

Uwa!! So Temperate♫

(Yes, I know. The '♫' isn't italic. Or in 8-Bit Operator.)

First up is an often overlooked track, Uwa!! So Temperate♫, which plays on the controller config screen in Spring and Fall.

In this example, the backing melody of track follows and arpeggiates this chord progression:
G#maj7 -> G#maj7 -> F#maj7 -> F#maj
G#maj7 -> G#maj7 -> Bmaj7 -> Bmaj

'Home'

Next up is a leitmotif on its own, 'Home'.

('Home' as heard in Home (Music Box))

'Home', as discussed on another page of this index, is a leitmotif used in relation to the Dreemurr household, as heard in tracks such as Home and You Can Always Come Home.

In these contexts, the arpeggiated Esus4 chord is used to instill a feeling of comfort, warmth, and nostalgia.
